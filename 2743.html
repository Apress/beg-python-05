Errata: Beginning Python (ISBN 1-59059-519-X)

Note: These errata refer to the first printing of the book. Some have been corrected in later printings.

Chapter 1

On page 20, the reader is instructed to put the python.exe file in the PATH environment variable to execute Python from the command line on Windows. Rather than doing this, the directory containing the Python executable should be added to the PATH variable.

On page 21 ... 

#/usr/bin/python2

... should be ... 

#!/usr/bin/python2

Chapter 5

On page 90, last two lines ...
>>> x is y
True

... should be ... 

>>> x is y
False

Chapter 7

On page 147, all uses of larch should be lark.

Chapter 9

On page 183 ...

Chapter 10

On page 228 ...

a < n < b

... should be ...

a ≤ n < b

Chapter 12

On page 277 ... 

vbox.Add(contents, proportion=1, flag=wx.EXPAND | wx.LEFT | wx.LEFT | wx.LEFT, border=5)

... should be ... 

vbox.Add(contents, proportion=1, flag=wx.EXPAND | wx.LEFT | wx.BOTTOM | wx.RIGHT, border=5)

Chapter 13

In Listing 13-1, you may need to change ... 

import sqlite

... into ... 

from pysqlite import dbapi2 as sqlite

Note that in Python 2.5, the Pysqlite wrapper is included as the sqlite3 module.
